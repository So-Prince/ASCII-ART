What is ASCII Art?

ASCII art is a technique of creating images using text characters instead of pixels.

The term comes from ASCII (American Standard Code for Information Interchange), which is a system that assigns numbers to characters like:

A → 65
a → 97
0 → 48
# → 35

Originally, computers could only display text—not images—so developers used characters to simulate graphics.

🎨 Core Idea

ASCII art works by arranging characters in rows and columns to visually resemble shapes.

Example:

  *
 ***
*****
 ***
  *

This is just text—but your brain interprets it as a shape.

🧱 Building Blocks of ASCII Art
1. Characters as Pixels

Think of each character as a pixel.

Some commonly used characters:

Character	Use
#	Solid / filled areas
.	Light shading
*	Highlights
`/ \	`
@	Dense/dark regions
2. Density Matters

Different characters give different “weights”:

@  (very dark)
#  
*  
.  (very light)

This is how detailed ASCII images are made—by choosing characters based on intensity.

3. Fixed Width (Very Important)

ASCII art relies on monospace fonts (every character has equal width).

If spacing breaks, the image breaks.
